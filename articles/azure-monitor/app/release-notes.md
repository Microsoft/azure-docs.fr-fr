# <a name="release-notes---application-insights"></a>Notes de publication – Application Insights

Cette page indique où trouver des notes de publication détaillées concernant les mises à jour et les correctifs de bogues pour chaque Kit de développement logiciel (SDK) d’Application Insights.

## <a name="sdk"></a>Kit SDK 

* Kits de développement logiciel (SDK) .NET
    - Pour les versions 2.12 et ultérieures : [Kits de développement logiciel (SDK) .NET [y compris ASP.NET, ASP.NET Core et adaptateurs de journalisation]](https://github.com/Microsoft/ApplicationInsights-dotnet/releases) 
    - Pour les versions plus anciennes :
      - [Kit de développement logiciel (SDK) de serveur web ASP.NET](https://github.com/Microsoft/ApplicationInsights-server-dotnet/releases)
      - [Kit de développement logiciel (SDK) .NET](https://github.com/Microsoft/ApplicationInsights-dotnet/releases) 
      - [Adaptateurs de journalisation .NET](https://github.com/Microsoft/ApplicationInsights-dotnet-logging/releases)
      - [ASP.NET Core](https://github.com/Microsoft/ApplicationInsights-aspnet5/releases)
* [Java](https://github.com/Microsoft/ApplicationInsights-Java/releases)
* [JavaScript](https://github.com/microsoft/ApplicationInsights-JS/releases)
* [Exportateur Python d’Azure Monitor](https://github.com/census-instrumentation/opencensus-python/blob/master/contrib/opencensus-ext-azure/CHANGELOG.md)
