# <a name="china-content-delivery-with-azure-cdn"></a>Distribution de contenu en Chine avec Azure CDN

Azure Content Delivery Network (CDN) Monde peut distribuer du contenu aux utilisateurs en Chine à partir de points de présence proches de la Chine ou de tout point de présence offrant les meilleures performances aux requêtes provenant de Chine. Toutefois, si la Chine est un marché important pour vos clients et si vous avez besoin de rapidité, il est préférable d’utiliser Azure CDN Chine.

Azure CDN Chine est différent d’Azure CDN Monde dans la mesure où il distribue du contenu à partir de points de présence situés en Chine, en partenariat avec plusieurs fournisseurs locaux. Pour être conforme à la réglementation en vigueur en Chine, vous devez souscrire un abonnement distinct pour utiliser Azure CDN Chine, et vos sites web doivent disposer d’une licence ICP. L’activation et la gestion de la distribution de contenu à l’aide du portail et de l’API sont les mêmes dans Azure CDN Monde et Azure CDN Chine.

## <a name="comparison-of-azure-cdn-global-and-azure-cdn-china"></a>Comparaison entre Azure CDN Monde et Azure CDN Chine

Azure CDN Monde et Azure CDN Chine proposent les fonctionnalités suivantes :

- Azure CDN Monde :

     - Portail : https://portal.azure.com  

     - Effectue la distribution de contenu en dehors de la Chine

     - Quatre niveaux tarifaires : Microsoft Standard, Verizon Standard, Verizon Premium et Akamai Standard

     - [Documentation](./index.yml)

- Azure CDN Chine :

     - Portail : https://portal.azure.cn

     - Effectue la distribution de contenu au sein de la Chine

     - Deux niveaux tarifaires : Standard et Premium

     - [Documentation](https://docs.azure.cn/en-us/cdn/)
