# <a name="azure-powershell-samples-for-azure-data-share"></a>Exemples Azure PowerShell pour Azure Data Share

Le tableau suivant comprend des liens vers des exemples de scripts Azure PowerShell pour Azure Data Share.

|Exemples PowerShell|Description|
|---|---|
|[Créer un compte Data Share](scripts/powershell/create-new-share-account-powershell.md)| Ce script PowerShell crée un nouveau compte Data Share. |
|[Créer un partage Data Share](scripts/powershell/create-new-share-powershell.md)| Ce script PowerShell crée un partage Data Share. |
|[Ajouter des jeux de données à un partage](scripts/powershell/add-datasets-powershell.md)| Ce script PowerShell ajoute un jeu de données blob à un compte Data Share existant. |
|[Définir et afficher les synchronisations d’un partage](scripts/powershell/set-view-synchronizations-powershell.md)| Ce script PowerShell crée et affiche les synchronisations d’un partage de données existant. |
|[Créer une invitation Data Share](scripts/powershell/create-share-invitation-powershell.md)| Cet exemple de script PowerShell crée une invitation pour accéder à un partage de données existant. |
|[Afficher les invitations de partage envoyées](scripts/powershell/view-sent-invitations-powershell.md)| Cet exemple de script PowerShell récupère la liste des invitations envoyées à partir d’un partage de données existant. |
|[Accepter les invitations de partage reçues](scripts/powershell/accept-share-invitations-powershell.md)| Cet exemple de script PowerShell accepte une invitation de partage de données.|
|[Afficher les détails d’un partage de données](scripts/powershell/view-share-details-powershell.md)| Cet exemple de script PowerShell répertorie et récupère les détails des partages de données. |
|[Surveiller l’utilisation des données partagées](scripts/powershell/monitor-usage-powershell.md)| Cet exemple de script PowerShell surveille l’utilisation des données partagées envoyées. |
|[Créer et afficher des déclencheurs d’instantanés](scripts/powershell/create-view-trigger-powershell.md)| Cet exemple de script PowerShell crée les déclencheurs d’instantanés d’un partage.
